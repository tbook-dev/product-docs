# View and Track the Data

### Asset Details

Click on the SBT card in the Incentive Assets list to view the deployed SBT information and status.

![](/img/zdUXIzv5DShxlQe7c5G4.png)

### Track User Engagement

Click on the `Track Engagement` button of the SBT DeepLink card with an **Ongoing** or **Completed** status to view the user participation details.

![](/img/nFO9YuDftshYZB5VUhit.png)
