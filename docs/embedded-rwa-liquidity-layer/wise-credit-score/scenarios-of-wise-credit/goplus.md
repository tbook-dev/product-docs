# GoPlus

A notable highlight in TBook's development is its multifaceted collaboration with GoPlus, an industry level security infrastructure. This showcases TBook's versatility and impact.

GoPlus is an open, permissionless and user-driven modular user security layer for Web3. Its main product SecWareX provides personalized security protection for users, ensuring your digital assets are not exposed to risks.

TBook has provided an all around service to GoPlus and has driven the partnership towards success.

## Empowering SecwareX with TBook's Incentive Layer

TBook's Incentive Layer has been the default choice for GoPlus to execute their customer faced incentive programs. TBook also empowers SecwareX with data powered user segmentation, helping GoPlus accurately target the most suitable incentive audience.

## Tokenomics Optimizations

Leveraging the notification capabilities of the TBook Incentive SDKs, GoPlus was able to fully exploit the incentive infrastructure to better understant its community.

TBook SDK is capable of identifying users and form user data feedback through real time call backs. User data from incentive passports can be feeded back to GoPlus server in real time, which empowers GoPlus to constantly evolve and optimize their tokenomics and commercial strategies. With TBook, GoPlus was able to balance the incentive demands among different incentive targets: investors, community, exchange, liquidity providers etc.
